# PiHardwareInfo
Get Raspberry Pi hardware info from /proc/cpuinfo


# Try
```bash
curl -s https://raw.githubusercontent.com/rexzhang/pi-hardware-info/master/pi_hardware_info.py | python3
```

# Install
```bash
pip install PiHardwareInfo
```
